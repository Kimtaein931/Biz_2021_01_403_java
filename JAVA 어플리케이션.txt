JAVA 어플리케이션

System.out.println(30 + 40);

CPU
제어장치(Controller)
ALU(Arithmetic logic unit)

메모리(Memory, 기억장치)
		30
		40
		70

- 30이라는 값을 CPU로 이동시키고, 40이라는 값을 CPU로 이동 시키고 
	(CPU는 기억장치 어딘가에 저장)
- (+)를 만나면 산술논리연산장치(ALU)로 보내 30과 40을 덧셈을 시킨 뒤 
- 다시 제어장치로 알려주고 결과값을 기억장치 어딘가에 저장 후
- out 명령어로 보내고 out에선 System으로 보냄(결과값 출력)
- 컴퓨터 기억장치의 특징 : 본래 있던 값에 다른 값이 들어온다면 무조건 사라짐
						(저장된 위치가 전부 다르다.)
컴파일 : 사람의 언어를 기계어로 바꿔주는 것

adress(주소) : 컴파일러는 기억장치에 저장되어 있는 유일한(고유한) 주소값을 가지고 찾음
